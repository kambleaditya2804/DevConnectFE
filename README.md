**DevConnect** is a MERN stack application that allows developers to **build profiles, explore peers, and establish connections**. This repository contains the **frontend**, built with React and Redux Toolkit, which communicates with the [DevConnect Backend](https://github.com/AdityaKambleV/DevConnect-backend).

## Tech Stack
- **Frontend:** React.js, Vite, Tailwind CSS  
- **State Management:** Redux Toolkit  
- **Backend:** [DevConnect Backend](https://github.com/kambleaditya2804/DevConnectBE)  
- **Authentication:** JWT-based login and signup  
- **Database:** MongoDB (handled by backend)
